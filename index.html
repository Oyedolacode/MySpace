About

Holla, i'm Oyedola Oluwasegun, a Front End Developer && UI/UX designer based in Lagos,Nigeria. I'm passionate about technology and constant learning.

I enjoy helping people improve the way they work using web applications,responsive website and design.

My journey first started with HTML and CSS in January, 2020. Then i met javascript by March at an internship HNG7 that halted my Dev journey when I discovered JavaScript was too hard, certainly because there was no adequate guidance.

In June 1, i found #100DaysOfCode hashtag and FreeCodeCamp, where i consistently put my time, never gave up this time. Since then I built 10 websites, 10 vanilla JavaScript projects, and earned 8 KYU Badge on CodeWars

In August 10, 2020, I met with another amazing love of my life called figma #figma , ever since i have not looked back to the design World again with the help of the figma community.

Now I'm more goal oriented, focused and hungry than ever before. I'm currently learning React.js, PHP, django, Photoshop and i'm going to build more projects.
